# pc_jagcrypt
A PC port of the Atari Jaguar Cartridge Encryption program.

## Versions
Windows 32 and 64 bits.<br>
MSYS2-MinGW32/64.<br>
MinGW32.<br>
Cygwin64.<br>
*nix.

## Supported projects
Visual Studio 2019 project.<br>
Makefile.
